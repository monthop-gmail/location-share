# คู่มือผู้ใช้ — แชร์ตำแหน่งเพื่อน

แอปแชร์ตำแหน่งแบบเรียลไทม์ เปิดจากมือถือแล้วเห็นตำแหน่งเพื่อนทันที

**เปิดแอป:** https://location-share.pages.dev

---

## 1. ใส่ชื่อเล่น

เปิดแอปแล้วพิมพ์ชื่อเล่นในช่อง **"ใส่ชื่อเล่นของคุณที่นี่..."**

- ชื่อจะถูกจำไว้ในเครื่อง ไม่ต้องใส่ใหม่ทุกครั้ง
- เพื่อนจะเห็นชื่อนี้บนแผนที่

---

## 2. แชร์ตำแหน่ง

1. กดปุ่ม **"เริ่มแชร์ตำแหน่ง"** (สีเขียว)
2. เบราว์เซอร์จะถามอนุญาตใช้ GPS → กด **"Allow"**
3. ตำแหน่งจะอัปเดตอัตโนมัติ ปุ่มเปลี่ยนเป็นสีแดง **"หยุดแชร์ตำแหน่ง"**
4. กดปุ่มแดงอีกครั้ง → หยุดแชร์ (ตำแหน่งถูกลบออก)

> ตำแหน่งจะหมดอายุอัตโนมัติหลัง 15 นาที ถ้าไม่อัปเดต

---

## 3. ดูตำแหน่งเพื่อน

เปิดแอปก็เห็นตำแหน่งเพื่อนทันที ไม่ต้องกดอะไร

### รายชื่อเพื่อน (ด้านล่างแผนที่)
- **จุดเขียว** = online (อัปเดตภายใน 15 นาที)
- **จุดเทา** = offline
- **กดชื่อเพื่อน** → แผนที่ zoom ไปหาเพื่อนคนนั้น
- **กด 🚗** → เปิด Google Maps นำทางไปหาเพื่อน

### ค้นหาเพื่อน
- พิมพ์ชื่อในช่อง **"ค้นหาชื่อเพื่อน..."**
- พิมพ์ `online` → ดูเฉพาะคนที่กำลังแชร์
- พิมพ์ `offline` → ดูเฉพาะคนที่ไม่ได้แชร์

### Fit Zoom
- กดปุ่ม **"ดูเพื่อนทุกคน (Fit Zoom)"** → แผนที่ zoom ให้เห็นทุกคน

---

## 4. ห้อง (Room)

สร้างห้องแยกสำหรับแต่ละกลุ่ม เพิ่ม `#ชื่อห้อง` ต่อท้าย URL

### ตัวอย่าง
| URL | ห้อง |
|-----|------|
| `https://location-share.pages.dev` | ห้องหลัก (main) |
| `https://location-share.pages.dev/#ทีมA` | ห้อง "ทีมA" |
| `https://location-share.pages.dev/#trip2026` | ห้อง "trip2026" |

- แต่ละห้องเห็นแค่คนในห้องเดียวกัน
- ส่งลิงก์ห้องให้เพื่อน → เปิดลิงก์เดียวกัน → เห็นกันทันที
- รองรับชื่อห้องภาษาไทย

---

## 5. วาดเส้นทาง

วาดเส้นทางบนแผนที่ เช่น เส้นทางเดินทาง จุดนัดพบ

1. กดปุ่ม **"วาดเส้นทาง"** (ปุ่มเปลี่ยนเป็นสีส้ม)
2. **กดจุดบนแผนที่** ทีละจุด ตามเส้นทางที่ต้องการ
3. **กดจุดสุดท้ายซ้ำ** เพื่อจบเส้นทาง
4. ใส่ชื่อเส้นทาง → กด OK
5. เส้นทางปรากฏบนแผนที่ทันทีสำหรับทุกคนในห้อง

### จัดการเส้นทาง
- รายการเส้นทางแสดงด้านล่างรายชื่อเพื่อน
- **กดชื่อเส้นทาง** → zoom ไปดูเส้นทาง
- **กด ✕** → ลบเส้นทาง

### ยกเลิกระหว่างวาด
- กดปุ่ม **"ยกเลิกวาด"** (สีส้ม) → ออกจากโหมดวาด

---

## 6. ปักหมุดจุดหมาย

ปักหมุดจุดหมาย เช่น ปลายทาง จุดพัก ปั๊มน้ำมัน ให้ทุกคนในห้องเห็น

1. กดปุ่ม **"ปักหมุดจุดหมาย"** (ปุ่มเปลี่ยนเป็นสีม่วง)
2. **กดจุดบนแผนที่** ที่ต้องการปักหมุด
3. ใส่ชื่อจุดหมาย + เลือก icon
4. กด **"บันทึก"** → หมุดปรากฏบนแผนที่ทันที

### icon ที่เลือกได้
| Icon | ความหมาย |
|------|----------|
| 📍 | ทั่วไป |
| 🏁 | ปลายทาง |
| ☕ | จุดพัก/ร้านกาแฟ |
| ⛽ | ปั๊มน้ำมัน |
| 🅿️ | ที่จอดรถ |
| 🏨 | โรงแรม/ที่พัก |

### จัดการหมุด
- รายการหมุดแสดงด้านล่าง
- **กดชื่อหมุด** → zoom ไปดู
- **กด 🚗** → เปิด Google Maps นำทางไปจุดหมาย
- **กด ✕** → ลบหมุด

### ยกเลิกระหว่างปักหมุด
- กดปุ่ม **"ยกเลิกปักหมุด"** (สีม่วง) → ออกจากโหมด

---

## 7. Tips

- **รองรับ:** LINE Browser, Chrome, Safari
- **ชื่อเล่น** จำไว้ในเครื่อง เปลี่ยนได้ตลอด
- **ตำแหน่ง** อัปเดตอัตโนมัติทุกครั้งที่ GPS เปลี่ยน
- **ไม่ต้องลงทะเบียน** ไม่ต้อง login — เปิดลิงก์ก็ใช้ได้เลย
- **เปิดหลายห้อง** ได้พร้อมกันในหลาย tab

---

## FAQ

**Q: หมุดจุดหมายหายไป?**
> หมุดจะอยู่จนกว่าจะมีคนลบ (ไม่หมดอายุเหมือนตำแหน่ง GPS)



**Q: ทำไมไม่เห็นตำแหน่งเพื่อน?**
> เพื่อนต้องเปิดแอปและกด "เริ่มแชร์ตำแหน่ง" ด้วย + ต้องอยู่ห้องเดียวกัน

**Q: GPS ไม่ทำงาน?**
> ตรวจสอบว่าอนุญาตให้เบราว์เซอร์เข้าถึงตำแหน่งแล้ว (Settings → Site permissions → Location)

**Q: ตำแหน่งเพื่อนหายไป?**
> ตำแหน่งหมดอายุหลัง 15 นาที ถ้าเพื่อนปิดแอปหรือหยุดแชร์

**Q: เปลี่ยนห้องยังไง?**
> เพิ่ม `#ชื่อห้อง` ต่อท้าย URL แล้วกด Enter หรือเปิดลิงก์ใหม่
